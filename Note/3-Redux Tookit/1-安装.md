# 安装
## NPM方式
```js
// JS模板
npx create-react-app my-app --template redux 
// TS模板
npx create-react-app my-app --template redux-typescript  @types/react-redux
// 已有项目中添加
npm install @reduxjs/toolkit @types/react-redux
```
## Yarn方式
```js
// JS模板
yarn create react-app my-app --template redux 
// TS模板
yarn create react-app my-app --template redux-typescript
// 已有项目中添加
yarn add @reduxjs/toolkit react-redux
yarn add @reduxjs/toolkit @types/react-redux
```
